# Overview

GridMaster is a competitive deckbuilder played on a 6x3 grid.

# Development Environment

This game is made in Godot with GDScript, using the plugins GodotFirebase, HTTPSSEClient, and Supabase to enable Firebase and Supabase integration, as well as LAN functionality.
